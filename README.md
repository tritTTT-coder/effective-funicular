NixNova — это легковесный и стильный аудиоплеер, написанный на Python. Проект имеет модульную архитектуру, разделенный на логику обработки звука (Core) и кастомизируемый графический интерфейс (UI).🚀 Особенности проектаАудио-движок: Стабильное воспроизведение аудиофайлов с низким потреблением ресурсов.Менеджер плейлистов: Удобное управление музыкальными треками, создание и сохранение списков воспроизведения.Кастомные темы: Встроенная поддержка тем оформления для настройки внешнего вида плеера под себя.Чистая архитектура: Код разделен на независимые модули (core, ui, utils), что упрощает его доработку и поддержку.
